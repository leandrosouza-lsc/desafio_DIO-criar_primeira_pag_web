História da Web

voltar

A história da web nos leva a criação dos primeiros computadores como o Mark I e Colossus, depois disso, com a evolução da tecnologia passamos para computadores menores até chegarmos aos computadores pessoais que temos atualmente. A internet surge com a nescessidade de comunicação, tivemos a primeira conexão estabelecida via Arpanet e décadas depois surge o conceito de TCP/IP e o primeiro navegador. Vários nomes fazem parte desta história, dentre eles: Grace Murray Hopper, Tim Berners-Lee, Bill Gates e Steve Jobs.

O que são Clients

voltar

Podemos entender como Clients todo dispositivo que esta conectado à rede e esta consumindo as informações requisitadas. O navegador é o programa utilizado para abrir e executar os arquivos requisitados. Os navegadores permitem ao usuário navegar pelas aplicações web que são soluções de comunicação entre Client e Server sem a nescessidade de instalação na máquina do usuário. Hoje em dia a internet esta cada vez mais presente nos dispositivos móveis, que apresentam uma crescente exponencial de usuários

O que são Servers

Podemos entender como Servers tudo que esta funcionando no ambiente do Servidor. Há diferentes tipos de servidores, servidores de: arquivos, segurança, streaming, e-mail, web. Os servidores possuem tanto softwares(programas) como hardwares(parte física). Para se hospedar um site é preciso o registro de um domínio e uma armazenagem em um servidor.

Linguagem de programação

Linguagem de programação é uma linguagem onde é possível efetuar códigos para que seja feito uma instrução ao computador. Há diferentes tipos de linguagens, as que são utilizadas para o Client Side e para o Server Side. HTML não é uma linguagem de programação, e sim apenas uma linguagem de marcação de texto.
